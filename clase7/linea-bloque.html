Elementos de linea vs de bloque

Elementos de línea

Los elementos de línea se presentan inline o sea uno al lado del otro.
Estos elementos no aceptan ancho ni altura

Elementos de bloque

Los elementos del bloque consumen todo el ancho del contenedor, o sea que se ubican uno debajo del otro
Éstos si aceptan ancho y altura
